# Mickey's QR Menu Project (V6)

## Project Status
Current Version: 1.6.0
State: V6 Features Implemented (Sorting, Design Settings, Pagination, Multi-language)

## Key Features
- **Drag & Drop Sorting**: Categories and Allergens can be reordered in the Admin Panel.
- **Design Settings**: Enhanced visual customization including fonts, layout width, and footer colors.
- **Languages**: Frontend language switcher (TR/EN).
- **Pagination**: Admin product list uses pagination (20 items/page).
- **Bulk Actions**: Bulk category update and bulk delete.

## File Structure
- `mickeys-qr-menu.php`: Main plugin file.
- `admin-view.php`: HTML/PHP for the admin interface.
- `admin-script.js`: Logic for the admin panel (AJAX, Sortable, Settings).
- `style.css`: Frontend styles.
- `script.js`: Frontend logic.
- `menu-data.js`: (Optional) Shared data structure.
- `admin-style.css`: Admin panel visuals.

## How to Continue
1. Unzip `MickeysQRMenu_Source_v6.zip` or open the folder `mickeys-qr-menu-wp`.
2. Edit files as needed.
3. Use the providing packaging scripts (e.g., `create_zip_tar.ps1`) to create a WordPress-ready zip file.
