Dinamik QR Menü ve Yönetim Paneli eklentisi.

== Açıklama ==

Bu eklenti, restoranınız için dinamik bir QR menü sistemi ve bu menüyü yönetebileceğiniz bir admin paneli sunar.

Özellikler:
*   Kategorilere ayrılmış ürün listesi
*   Admin panelinden ürün ekleme, düzenleme, silme
*   Excel ile toplu ürün yükleme/indirme
*   Fiyat yönetimi (toplu güncelleme, yuvarlama)
*   Alerjen yönetimi
*   Banner yönetimi

== Kurulum ==

1.  Bu klasörü `.zip` formatında sıkıştırın.
2.  WordPress Admin paneline gidin -> Eklentiler -> Yeni Ekle -> Eklenti Yükle.
3.  Zip dosyasını seçin ve yükleyin.
4.  Eklentiyi etkinleştirin.

== Kullanım ==

1.  **Admin Paneli:** WordPress sol menüsünde "QR Menu Panel" sekmesine tıklayarak yönetim paneline ulaşabilirsiniz.
2.  **Menüyü Gösterme:** Menüyü web sitenizde göstermek istediğiniz sayfaya `[mickeys_qr_menu]` kısa kodunu (shortcode) ekleyin.

== Notlar ==

*   Eklenti etkinleştirildiğinde varsayılan menü verileri otomatik olarak yüklenir.
*   Veriler WordPress veritabanında (wp_options tablosunda) saklanır.
