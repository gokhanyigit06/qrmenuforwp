== Description ==

**Menu QR Pro** is the ultimate digital menu solution for restaurants, cafes, bars, and food businesses. Create stunning, interactive menus that customers can access via QR codes on their smartphones.

= Key Features =

* **Easy Menu Management** - Intuitive admin panel to add, edit, and organize your menu items
* **QR Code Ready** - Generate QR codes for contactless menu viewing
* **Multi-Language Support** - Built-in Turkish and English translations, easily add more languages
* **Beautiful Themes** - 15+ pre-designed themes (Modern Bistro, Ocean Blue, Tokyo Street, etc.)
* **Category Management** - Organize items into customizable categories with icons and images
* **Allergen Information** - Display allergen warnings to keep customers informed
* **Price Management** - Bulk price updates, percentage changes, and smart rounding
* **Excel Import/Export** - Manage menu in Excel and bulk import
* **Product Variants** - Support for different sizes, options (Small/Medium/Large)
* **Banner Slider** - Promote special offers and campaigns
* **Responsive Design** - Perfect on all devices (mobile, tablet, desktop)
* **Drag & Drop Sorting** - Easy reordering of categories and items
* **Visual Customization** - Fonts, colors, layout width, border radius, and more
* **Search & Filter** - Quick product search and category filtering
* **No Coding Required** - Simple shortcode integration

== Installation ==

= Automatic Installation =

1. Log in to your WordPress admin panel
2. Go to Plugins → Add New
3. Search for "Menu QR Pro"
4. Click "Install Now" and then "Activate"

= Manual Installation =

1. Download the plugin ZIP file
2. Log in to WordPress admin panel
3. Go to Plugins → Add New → Upload Plugin
4. Choose the ZIP file and click "Install Now"
5. Activate the plugin

= After Activation =

1. Go to "QR Menu Panel" in your WordPress admin sidebar
2. Add your first category
3. Start adding menu items
4. Customize visual settings
5. Use shortcode `[mickeys_qr_menu]` to display menu on any page

== Frequently Asked Questions ==

= How do I display the menu on my website? =

Use the shortcode `[mickeys_qr_menu]` on any page or post.

= Can I change the menu language? =

Yes! The plugin automatically detects WordPress language. Currently supports Turkish and English.

= Is it mobile-friendly? =

Absolutely! The menu is fully responsive and optimized for mobile devices.
